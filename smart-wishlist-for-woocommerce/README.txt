=== Smart Wishlist Plugin For WooCommerce is an extension designed specifically for WooCommerce. ===

== Description ==

The "Smart Wishlist" plugin for WooCommerce is designed to enhance the wishlist functionality on your WooCommerce store. 

It provides advanced features and customization options to improve the wishlist experience for your customers.

=== Basic features ===

== The basic features of the free version of the "Smart Wishlist" plugin for WooCommerce may include:==

	* Select Wishlist Button Type Options: Users can choose the style or type of button to display for adding products to the wishlist. 
	This allows customization based on preferences or design requirements.

	* Customize Add to Wishlist Button Text: The plugin provides the ability to customize the text displayed on the "Add to Wishlist" button. 
	This allows users to personalize the language or call-to-action for adding items to their wishlist.

	* Select Wishlist Page and Create Wishlist Page: Users can select an existing page or create a dedicated wishlist 
	page where customers can view and manage their wishlists. This helps centralize wishlist functionality for easy access.

	* Show "Add to Wishlist" Button on Shop Page: The plugin enables the display of the "Add to Wishlist" button directly on the Shop page.
	This provides a convenient way for customers to add products to their wishlist without navigating away from the main shop page.

	* Select View Button Type Options: Users can choose the style or type of button to display for viewing the wishlist. 
	This allows customization to match the design or preferences of the website.

	* Customize "View Wishlist" Text: The plugin allows customization of the text displayed on the "View Wishlist" button. 
	This gives users the flexibility to customize the wording for accessing and viewing their wishlist.

	* Customize "Add to Wishlist" Icon: Users can customize the icon associated with the "Add to Wishlist" button. 
	This provides the ability to choose a specific icon that aligns with the website's design or visual identity.

	* Customize Columns in Wishlist Table: Users can customize the columns displayed in the wishlist table. 
	This allows for personalization and customization of the information displayed for each wishlist item.

	* Share Wishlist on Social Channels: The plugin provides social sharing options, allowing customers to share their 
	wishlist with others through popular social media platforms such as WhatsApp, Facebook, Gmail, Instagram, Pinterest, Twitter, and LinkedIn. 
	This facilitates sharing desired products with friends, family, or social networks.

	* CSS Styles Options allow you to customize the appearance of the wishlist buttons in the "Smart Wishlist" plugin for WooCommerce. Based on the provided options

These features provide basic functionality for managing wishlists, customizing buttons and text, and enabling social sharing options. 

=== Premium Features ===

== Upcoming Premium Features for the "Smart Wishlist" plugin for WooCommerce include: ==

	
	* Wishlist Total Price: The plugin will calculate and display the total price of all the products in the wishlist. 
	This gives users a quick overview of the total value of the items they have added to their wishlist.

	* Products Quantity: Users will be able to specify the desired quantity for each product in their wishlist. 
	This allows for more accurate tracking of desired quantities and helps with planning and decision-making.

	* Change Wishlist Table Layout: Users will have options to customize the layout and appearance of the wishlist table. 
	They can choose from different table styles, column configurations, and sorting options to suit their preferences.

	* Wishlist Show on My Account Page: The plugin will integrate the wishlist functionality into the "My Account" page of users. 
	This provides a central location for users to access and manage their wishlists alongside other account-related information.

	* Search Bar on Wishlist Page: Users will have the ability to search for specific products within their wishlist. 
	This makes it easier to locate desired items, especially when the wishlist contains a large number of products.
	
	* Multi Wishlist Creation: Enables users to create, manage, and personalize multiple wishlists, offering flexibility for organizing desired items, gifts, or goals, enhancing user experience and organization.
	
	These upcoming premium features enhance the functionality and customization options of the "Smart Wishlist" plugin, providing additional control, privacy, and convenience for both users and administrators.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of WooCommerce Order Coupon, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Smart Wishlist Plugin for WooCommerce" and click Search Plugins. Once you've found our Smart Wishlist Plugin for WooCommerce you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Smart Wishlist Plugin for WooCommerce and uploading it to your web server via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
